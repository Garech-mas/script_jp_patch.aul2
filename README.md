# AviUtl2 スクリプト日本語化パッチ
AviUtl2のスクリプトに含まれる英語パラメータ・項目名を日本語化し、可読性を向上させる言語ファイル（`.aul2`）です。

## ダウンロード・インストール
https://github.com/Garech-mas/script_jp_patch.aul2/releases/latest

上記のリンクから `script_jp_patch.au2pkg.zip` をダウンロードできます。

AviUtl2を起動し、ダウンロードしたZIPファイルをプレビュー画面にドラッグ＆ドロップすることで導入できます。

再起動後、メニューバー **設定** > **言語の設定** から、`スクリプト日本語化パッチ` を選択して適用してください。

[アクセスキー付き日本語リソース](https://github.com/Garech-mas/jp_akey.aul2/edit/main/README.md) を使用している場合、 `jp_akey_aul2_script_jp_patch.au2pkg.zip` ファイルを導入することで共存できます。
